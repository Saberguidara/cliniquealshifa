# Prompt — Landing Page Clinique Médicale

## Instruction pour l'agent

Crée une landing page professionnelle et moderne pour une clinique médicale privée appelée **"Clinique AlShifa"**, située à Sfax, Tunisie.

### Stack technique
- HTML5 sémantique
- CSS3 avec Tailwind CSS (via CDN)
- JavaScript vanilla (pas de framework)
- Responsive mobile-first
- Un seul fichier index.html + un fichier style.css + un fichier main.js

### Sections à créer (dans l'ordre)
1. **Header/Navbar** — Logo texte, liens de navigation (Accueil, Services, Médecins, Contact), sticky au scroll
2. **Hero** — Titre accrocheur, sous-titre, bouton "Prendre RDV", image ou illustration médicale (utilise une image Unsplash via URL)
3. **Services** — 6 cartes de services (Cardiologie, Pédiatrie, Radiologie, Chirurgie, Urgences, Laboratoire), icônes SVG simples
4. **Pourquoi nous choisir** — 3 arguments avec icônes (Équipe experte, Équipements modernes, Disponible 24h/24)
5. **Médecins** — 3 cartes médecins avec photo (Unsplash), nom, spécialité
6. **Témoignages** — 3 avis patients avec étoiles
7. **Contact** — Formulaire (Nom, Email, Message, bouton Envoyer), adresse, téléphone, Google Maps embed
8. **Footer** — Logo, liens, réseaux sociaux, copyright

### Design
- Couleurs : bleu médical (#1E40AF) + blanc + gris clair
- Police : Google Fonts "Inter"
- Animations CSS légères au scroll (fade-in)
- Hover effects sur les cartes et boutons

### Contraintes
- 100% responsive (mobile, tablette, desktop)
- Pas de dépendances npm, tout via CDN
- Code propre et commenté
- Valide W3C

### Livrable
- `index.html`
- `style.css`
- `main.js`
- `README.md` avec instructions d'installation et screenshot placeholder
